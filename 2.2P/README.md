## SIT725 Task 2.2P — Express + REST API (Add Two Numbers)

### Overview

This task sets up a simple Node.js + Express server that:

- Serves a static webpage from the `public/` folder
- Provides REST API endpoints, including an endpoint to add two numbers via URL (GET)

### Tech

- Node.js
- Express

### Project structure

- `server.js` — Express server (routes + static hosting)
- `public/` — static files served by Express
  - `index.html`
  - `scripts.js`

### Setup & run

From the repo root:

```bash
cd 2.2P
npm install
npm run dev
```
